## 🚀 Порядок действий

### 1. Создать оферту (10 минут)
```bash
# После создания oferta.pdf:
# Положить файл в папку public/
# Затем:
git add public/oferta.pdf
git commit -m "Add oferta.pdf"
git push origin main
```

Railway автоматически задеплоит обновление.

### 2. Проверить что всё работает (2 минуты)

Открыть в браузере:
- ✅ https://descro-production.up.railway.app (главная)
- ✅ https://descro-production.up.railway.app/#pricing (цены)
- ✅ https://descro-production.up.railway.app/requisites (реквизиты)
- ⚠️ https://descro-production.up.railway.app/oferta.pdf (оферта)
- ✅ https://descro-production.up.railway.app/payment/success
- ✅ https://descro-production.up.railway.app/payment/cancel

### 3. Подать заявку в ЮKassa (5 минут)

1. Зайти на https://yookassa.ru
2. Нажать "Подключиться"
3. Выбрать "Платежи на сайте"
4. Заполнить форму данными из этого файла (см. выше)
5. Отправить заявку

### 4. Дождаться одобрения (1-3 дня)

ЮKassa проверит:
- ✅ Сайт работает
- ✅ Есть цены и описания услуг
- ✅ Есть реквизиты с ИНН
- ✅ Есть оферта
- ✅ Есть контакты

### 5. Получить ключи и настроить

После одобрения:
1. Получить Shop ID и Secret Key
2. Добавить в Railway Variables
3. Настроить webhook
4. Протестировать оплату

Подробная инструкция: `YOOKASSA_SETUP.md`
