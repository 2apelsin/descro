## 🎯 Как протестировать

### 1. Откройте сайт
```
http://localhost:3000
```

### 2. Откройте модалку
- Нажмите кнопку "Войти" в хедере
- Или нажмите на любую кнопку которая требует авторизации

### 3. Проверьте режим Magic Link
1. Выберите "Magic Link" (по умолчанию)
2. Введите email
3. Нажмите "Получить ссылку"
4. Должно появиться: "✅ Ссылка отправлена на email"
5. Проверьте почту (и папку Спам)
6. Кликните на ссылку

### 4. Проверьте режим с паролем
1. Выберите "С паролем"
2. Введите email
3. Введите пароль (минимум 6 символов)
4. Проверьте счётчик: "0/6" → "6/6" ✓
5. Нажмите глазик 👁️ - пароль должен показаться
6. Нажмите ещё раз - пароль скроется
7. Нажмите "Войти / Создать аккаунт"

### 5. Проверьте Telegram Login
1. Нажмите на кнопку Telegram
2. Должен открыться виджет Telegram
3. Авторизуйтесь через Telegram
4. Должно перезагрузить страницу с авторизацией

## 🐛 Возможные проблемы

### Модалка не открывается
- Проверьте консоль браузера (F12)
- Убедитесь что переменные окружения установлены:
  ```bash
  NEXT_PUBLIC_SUPABASE_URL=...
  NEXT_PUBLIC_SUPABASE_ANON_KEY=...
  ```

### Telegram Login не работает
- Проверьте `NEXT_PUBLIC_TELEGRAM_BOT_USERNAME` в .env.local
- Убедитесь что домен настроен в @BotFather (`/setdomain`)

### Ошибка "Invalid login credentials"
- Это нормально при первом входе
- Система автоматически создаст аккаунт
- Проверьте email для подтверждения

### Множественные экземпляры GoTrueClient
- Это исправлено! Теперь используется singleton
- Если всё ещё видите - очистите кеш браузера
