# Descro — AI-генератор описаний для маркетплейсов

Полнофункциональный лендинг с рабочей генерацией, эффектами и конверсионными элементами.

## 🚀 Быстрый старт

```bash
npm install
npm run dev
```

Откройте http://localhost:3000

## 🎬 Как работает

1. **Заполните форму** — есть быстрые примеры
2. **Нажмите "Сгенерировать"** — появится скелетон
3. **Смотрите прогресс** — "Анализируем товар... 30%"
4. **Результат печатается** — эффект печатной машинки
5. **Копируйте** — каждый блок или всё сразу

## 🎁 Конверсионные элементы

- **Таймер/скидка** — "Первым 10 → 199₽"
- **Отзывы** — 2 мини-отзыва на Hero
- **Социальное доказательство** — "150+ продавцов"
- **FAQ** — 4 частых вопроса
- **Цена на видном месте** — в Hero и Pricing

## 📊 Метрики

Отслеживайте:
- Конверсия в генерацию (visitors → generated)
- Конверсия в Pro (free → paid)
- Среднее время до первой генерации
- Процент повторных визитов

## 🔧 Настройка

### Подключение OpenAI (опционально)

1. Добавьте в `.env.local`:
```env
OPENAI_API_KEY=sk-your-key
```

2. Обновите `app/api/generate/route.ts`

### Изменение лимита

В `components/demo-form.tsx`:
```typescript
const DAILY_LIMIT = 3 // Измените на нужное значение
```
