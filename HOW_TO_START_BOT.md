# 🤖 Как запустить Telegram бота

## Быстрый старт (3 команды)

Открой терминал и выполни:

```bash
cd telegram-bot
npm install
node bot.js
```

Всё! Бот запущен и работает.

---

## Подробная инструкция

### Шаг 1: Открыть терминал

**Windows:**
- Нажми `Win + R`
- Введи `cmd` или `powershell`
- Нажми Enter

**Или:**
- Открой папку проекта в проводнике
- В адресной строке введи `cmd` и нажми Enter

### Шаг 2: Перейти в папку бота

```bash
cd telegram-bot
```

Если ты уже в корне проекта, эта команда перенесет тебя в папку с ботом.

### Шаг 3: Установить зависимости

```bash
npm install
```

Эта команда установит 2 библиотеки:
- `node-telegram-bot-api` — для работы с Telegram
- `jsonwebtoken` — для создания токенов авторизации

Установка займет 10-30 секунд.

### Шаг 4: Запустить бота

```bash
node bot.js
```

Ты увидишь:
```
🤖 Descro Telegram Bot started...
```

Это значит бот работает! ✅

## Проверка работы

1. Открой Telegram
2. Найди бота: @Telegagocod_bot
3. Отправь команду `/start`
4. Бот должен ответить с кнопкой "Войти на сайт"

## Что делать если не работает

### Ошибка: "npm не является внутренней командой"

**Решение:** Установи Node.js
1. Скачай с https://nodejs.org
2. Установи (выбери LTS версию)
3. Перезапусти терминал
4. Попробуй снова

### Ошибка: "Cannot find module"

**Решение:** Установи зависимости
```bash
npm install
```

### Ошибка сети при установке

**Решение 1:** Попробуй другой registry
```bash
npm install --registry=https://registry.npmjs.org/
```

**Решение 2:** Установи пакеты по отдельности
```bash
npm install node-telegram-bot-api
npm install jsonwebtoken
```

### Бот не отвечает в Telegram

**Проверь:**
1. Бот запущен? (должно быть сообщение "Bot started...")
2. Правильный бот? (@Telegagocod_bot)
3. Токен правильный? (в файле bot.js)

## Запуск на фоне (опционально)

### Windows

Создай файл `start-bot.bat`:
```batch
@echo off
cd telegram-bot
node bot.js
```

Запусти двойным кликом.

### Linux/Mac

```bash
cd telegram-bot
nohup node bot.js &
```

3. Подключи GitHub репозиторий
4. Укажи Root Directory: `telegram-bot`
5. Добавь переменные окружения:
