# ✅ Улучшено отображение PRO подписки

## Что изменилось

### 1. Дашборд (/dashboard)
Для PRO пользователей теперь показывается:
- ⭐ "PRO подписка активна"
- **∞ Безлимитные генерации** (крупным шрифтом)
- Дата окончания подписки
- Количество дней до окончания
- Карточки со статистикой:
  - ∞ Генераций в месяц
  - X Дней осталось

### 2. Шапка сайта (site-header)
- Добавлен бейдж **PRO** рядом с именем пользователя
- Бейдж с градиентом (зеленый → синий)
- Показывается только для активных PRO пользователей

### 3. Главная страница (demo-form)
- Для PRO: "✨ PRO активен — безлимитные генерации"
- Для Free: "Осталось X из 3 бесплатных генераций"

### 4. Generation Limiter
- Для PRO: "✨ PRO активен до [дата]"
- Для Free: "Осталось попыток сегодня: X"

## Как это выглядит

**Для PRO пользователей:**
- В шапке: `Имя [PRO]`
- В дашборде: большой блок с ∞ и "Безлимитные генерации"
- На главной: "✨ PRO активен — безлимитные генерации"

**Для Free пользователей:**
- В шапке: просто имя
- В дашборде: "Осталось сегодня: 3 из 3"
- На главной: "Осталось X из 3 бесплатных генераций"

## Тестирование

1. Купи подписку за 1₽ (тестовая цена)
2. Проверь что в шапке появился бейдж PRO
3. Зайди в /dashboard - увидишь ∞ Безлимитные генерации
4. На главной странице увидишь "PRO активен"
5. Генерации не будут ограничены

Все работает автоматически - система проверяет `pro_until` и если дата в будущем, показывает PRO статус.
